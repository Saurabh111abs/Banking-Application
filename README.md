# 🏦 Banking Application

A RESTful Banking Application built with **Java** and **Spring Boot**, designed to perform core banking operations.

## 🚀 Features
- Create and manage bank accounts
- Deposit & Withdraw funds
- Transfer money between accounts
- View account balance and transaction history

## 🛠️ Tech Stack
- **Java** (Spring Boot)
- **Spring Data JPA**
- **MySQL / H2 Database**
- **Maven**
- **REST API**

## ⚙️ How to Run

```bash
# Clone the repo
git clone https://github.com/Saurabh111abs/Banking-Application.git

# Navigate to project
cd Banking-Application

# Run using Maven
./mvnw spring-boot:run
```
